== Description ==
This Plugin To Show Your Video With Ads And Poster.
 Ajax Control.
 Easy To Make Video Posts.
 Quick and easily usage.


= Usage =

1. To get started, click on the menu New Vp > New v. 
2. Use Shortcode.


== Installation ==

1. Upload the 'V-player' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Add the shortcode to your page.
